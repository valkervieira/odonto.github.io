ENDODONTIA - RETRATAMENTO

Protocolo

  1. Desobturação: Limas + Largo 1,2 (guta antiga = 2+ sessões)
  2. Odontometria + PQM
  3. Medicação: Ca(OH)₂ (14 dias)
  4. Obturação
  5. Reabilitação: Pino + Coroa

Taxas de Sucesso

  • Cura: 87,5%
  • Cicatrização periapical: 64-93,7%
  • Sucesso geral: 78-86,4%
  • RR cirúrgico vs não-cirúrgico: ~igual (1,05)

Prevalência PA

  • Geral: 6,3%
  • Dentes tratados: 41% têm PA
  • Não tratados: 3,5% têm PA

PERIODONTIA

Índices Ideais

  • IPV < 10%
  • ISG < 10%

Classificação Periodontal

Estágios: I (inicial) → II (moderado) → III (severo) → IV (muito severo)
Graus: A (lenta) | B (moderada) | C (rápida)

Sondagem

  • PS: Profundidade (margem → fundo)
  • PI: Perda Inserção (JCE → fundo)

RAP vs RASUB

  • RAP: Raspagem supragengival
  • RASUB: Raspagem subgengival

MTA

Características

  • pH: 10,2 → 12,5 (antibacteriano)
  • Presa: 3-4h (completa 24h)
  • Libera: Ca(OH)₂ ≥ 3 meses
  • FDA: 1998

Usos

Perfurações (raiz/furca)
Capeamento pulpar direto
Pulpotomias
Reabsorções externas
Selamento apical

Mecanismo

pH alcalino → fosfatase alcalina → osteoblastos → hidroxiapatita

PRÓTESE TOTAL

Etapas

  1. Moldagem inicial (alginato)
  2. Moldagem funcional (ZOE)
  3. Placa base + roletes
  4. DVO + ASA + seleção dentes
  5. Prova
  6. Acrilização
  7. Entrega + proservação

Adaptação

  • Média: 78 dias (~2,5 meses)
  • Normal: 3-6 meses

O que AUMENTA tempo?

1ª prótese mandibular
Reabsorção posterior severa (70% tipo III-IV)
Uso irregular
Úlceras persistentes

Reabsorção Crista

  • Tipo III: 11-15mm (severo)
  • Tipo IV: ≤10mm (muito severo)

PRÓTESE PARCIAL REMOVÍVEL

Manutenção Periodontal

  • 3-6 meses: ↑ sobrevida significativo
  • 1x/ano ou sem: ↑ risco perda

Causas Perda Dentes Pilares

  • 42% doença periodontal
  • 30% cárie
  • 26% fratura

RESTAURAÇÕES

ONLAY (Cerômero)

Sessão 1:

  • Remove restauração
  • Preparo
  • Selamento imediato
  • Moldagem fio 00 + silicona
  • Provisório: Bioplic

Sessão 2: Prova

Sessão 3 - CIMENTAÇÃO:

  • Peça: Microjato + ác.fosfórico 37% + silano
  • Preparo: Ác.fosfórico 37% (10s) + adesivo
  • Cimento: Dual + foto 60s/face

COROA ZIRCÔNIA

6 Sessões:

  1. Desobturar 2/3 + moldagem pinjet
  2. Prova pino + cimentação RelyX U200 + provisório
  3. Ajuste preparo
  4. Moldagem final (fio 000 + silicona)
  5. Prova monobloco + registro
  6. Cimentação

COROA METALOCERÂMICA

Etapas principais:

  • Pino fibra vidro (RelyX U200)
  • Preparo + provisório
  • Moldagem relocalização
  • Prova coping
  • Aplicação cerâmica
  • Cimentação: Fosfato de zinco

PINOS

Fibra Vidro:
Módulo = dentina
Estético

Metálico:

Excelente estética

Camada porcelana = fraco
Alto chipping

ÚLTIMO: RESINA COMPOSTA

Maior fratura
Maior desgaste

CLAREAMENTO

Clareamento Caseiro

  • PC 10%: Baixa concentração
  • Uso: Noturno (durante sono)
  • Moldeiras: Personalizadas

Pastas Dessensibilizantes

EFICAZ:

  • PC alta concentração
  • PH consultório sessão única

INEFICAZ:

  • PC 10% (baixa concentração)
  • PH consultório 2 sessões

Após 72h: níveis similares

EROSÃO / LCNC

Tipos

  • Pires: Suaves, arredondadas
  • Cunha (V): Profundas, agudas (maior risco)

Progressão

  • Média: 1,50 ± 0,92 mm³/ano

Etiologia

  • Escovação horizontal vigorosa
  • Dieta ácida
  • Refluxo
  • Medicamentos xerostômicos
  • Oclusão: Controverso

PROSERVAR

Sem sintomas
Sem comprometimento

AUMENTO COROA CLÍNICA

Espaço Biológico = 3mm

  • 1mm inserção conjuntiva
  • 1mm epitélio juncional
  • 1mm sulco

Protocolo

  1. Anestesia
  2. Incisão intrasulcular (15C) - 1mm
  3. Descolamento (Molt)
  4. Remove granulação (Gracey 11-12, 13-14)
  5. Osteotomia (cinzel) - RESPEITAR 3mm
  6. Sutura (Nylon 4.0)
  7. Remove: 7-14 dias

Pós-Op

Aguardar 6-8 semanas para prótese
Clorexidina 0,12%

DICAS PROVA

Caso Endo

  • Tecnologias: TCFC, microscópio, localizador
  • Sucesso: 78-86,4%
  • RR ≈ 1,05

Caso Perio

  • Classificar: Estágio + Grau
  • IPV/ISG < 10%
  • RAP = supra, RASUB = sub

MTA

  • Usos: perfuração, capeamento, pulpotomia, reabsorção
  • pH 12,5 = antibacteriano
  • 24h presa

Prótese

  • DVO = Dimensão Vertical Oclusão
  • ASA = Articulador Semi-Ajustável
  • 78 dias adaptação
  • PPR: 3-6M manutenção

Restaurações

  • Zircônia > Dissilicato > Resina (bruxismo)
  • Selamento imediato
  • Pino ≥ 2/3
  • Reparo resina = conservador

Clareamento

  • PC 10% = não precisa pasta
  • Erosão: tratar causa antes estética

ACC

  • 3mm SEMPRE

  • Nylon 4.0
